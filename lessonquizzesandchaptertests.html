Test Package for Blackboard/Moodle

If you are using Blackboard or Moodle, you should download the following test package files. There are two options for download, depending on your import goal. Download the Blackboard Pool file if you want to import the questions to a pool in your LMS; download the Blackboard Test file if you want to import the questions as a test.

The files are provided in archive file (.zip) format for LMS import. Test packages for all chapters have been zipped for your convenience.
